# 研究動機與創作理念

一直以來想下載音樂時總要到Youtube找音樂、複製網 址，再到網站上貼上網址下載，尤其要下載多首歌時，重 複的動作更是繁瑣，十分耗時又不便。 在這次的作業中，我靈機一動，想到可以來做做看下載音 樂的bot!想做出一個既方便又好用的「音樂下載bot」。
